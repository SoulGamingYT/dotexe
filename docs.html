  • Invite DotExe using the invite button
  • Give administrator permissions (recommended)
  • Run .help to explore commands

🛡️ Moderation System

DotExe provides powerful moderation tools including bans, mutes, jail system, and real-time logging.

  • Use .ban, .kick, .mute for manual moderation
  • Use .jail for controlled restrictions
  • All actions are logged automatically
